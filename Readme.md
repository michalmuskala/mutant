# Mutant

Mutant is a program to evolve images using simulated annealing method.

## Dependancies

* SDL2,
* SDL2_image,
* libpng for PNG, etc.

## Compiling

Under unixes, i.e. ArchLinux:
```
$ git clone https://github.com/michalmuskala/mutant
$ cd mutant
$ yaourt -S SDL2 SDL2_image libpng
$ make
$ bin/mutant
```
